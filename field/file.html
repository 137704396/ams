# 如何选择 file 和 originfile 类型

  • file 适合文件上传是单独的接口，独立于表单
  • originfile 用于表单内文件上传需要通过二进制的方式根据表单一起提交上传，originfile 需要设置 contentType 为 multipart 才能正常上传
